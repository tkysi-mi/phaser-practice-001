# Phaser Supermarket Quest

Phaser 3 を使った、CEFR A2 向けの買い物タスク型英語学習ゲームです。

## 遊び方

- 5分以内に、未知の料理に必要な食材をスーパーで集めます。
- プレイヤーは `←` `→` `↑` `↓` で移動し、店員の近くで `SPACE` で会話します。
- 会話クイズは選択肢が 2〜5 個でランダムに出ます。
- クイズ正答で食材ヒントが解放されます。ただし現行実装では、未解放の食材でも近づけば回収できます。
- ミスで即失敗はありません。制限時間内に必要食材だけを揃え、余計な物を持たずにレジで会計できればクリアです。

## 実行方法

1. このフォルダでローカルサーバーを起動します（例: VS Code Live Server）。
2. `index.html` をブラウザで開きます。

> `file://` 直開きでも動く場合がありますが、ローカルサーバー経由を推奨します。
